# Changelog

本文档记录 `Coco Dense` 的关键版本变更，重点保留会影响接手开发、发布和排查问题的信息。

## [0.2.0] - 2026-06-21

### 头像功能

- 支持上传、裁剪、删除个人头像
- 头像显示在锁定页面（替换锁图标）和欢迎页（替换应用图标）
- 裁剪支持双指捏合缩放（以中心为锚点）、双指滚动平移、三指左右拖拽
- 修复头像删除后仍显示的跨窗口状态同步问题
- 修复锁定页面头像背景色残留问题

### 回收站

- 新增回收站功能，删除文件夹时条目移至未分组而非直接删除
- 回收站支持分页浏览（每页 5 条）
- 支持定时自动清理：1 天 / 3 天 / 7 天 / 15 天 / 1 月 / 6 月 / 1 年 / 不自动删除

### 设置页面优化

- 密码长度设置从「通用」移至「安全」页面，输入框改为紧凑布局
- 修复头像与二次确认卡片的 HTML 结构和背景分离
- 删除「启动时显示欢迎页」功能，默认显示欢迎页

### 锁定与解锁优化

- 修复指纹解锁进入主页时的页面闪动和卡顿
- 修复锁定页面指纹按钮文字逻辑问题
- 修复首次解锁时的过渡动画不流畅

### 备注区域

- 备注文字过多时限制最大高度并支持滚动，修复溢出问题

### Bug 修复

- 修复 `purgeExpiredDeletedEntries` 加载时未持久化的问题
- 修复 `createFolder` 缺少 `await persistVault()` 的问题
- 修复新建条目 `lastUsedAt` 应为空字符串
- 修复 `confirmWithin` 确认时 toast 未立即隐藏
- 修复新增条目时点击其他条例未提示保存的问题

---

## [0.1.0] - 2026-06-20

### 首个公开整理版本

- 完成仓库迁移并统一到 `is-coco/Coco_Dense`
- 整理 GitHub Release 发布链路
- 统一安装包命名为 `Coco.Dense-*`
- Release 仅保留 `.dmg`、`.zip`、`.exe` 三个主要安装包

### 核心功能

- 本地加密保险箱
- 主密码 + 数据钥匙解锁
- 条目增删改查
- 文件夹分组
- 标签与优先级筛选
- 拼音搜索
- WebDAV 同步
- 导入 / 导出保险箱
- 应用内检查更新

### 安全与解锁

- 主密码不在渲染层长期缓存
- 主进程会话持有解锁态
- 支持 Touch ID 解锁链路
- 支持安全问题找回主密码
- 支持剪贴板自动清理
- 支持自动锁定和手动锁定

### 同步修复

- 修复双向同步只同步条目、不同步文件夹的问题
- 增加对旧结构 `entries / folders` 的兼容读取
- 修复新装或本地清缓存后，空库可能误覆盖云端数据的问题
- 保留云端已有数据作为更安全的首次同步策略

### 锁定与窗口行为修复

- 修复手动锁定后窗口尺寸异常变化的问题
- 修复自动锁定会强制切前台的问题
- 自动锁定改为后台静默锁定

### 更新与发布修复

- 应用内更新地址改为新仓库 `is-coco/Coco_Dense`
- 更新页默认跳转新 Release 页面
- 修复构建阶段自动发布导致的 GitHub Token 报错
- GitHub Actions 升级到新的 action 主版本
- 发布流程改为构建与上传分离，避免 Release assets 混乱

### 文案与体验

- 将界面中的“生物识别”相关文案统一调整为 `Touch ID识别`
- 更新 README，补充功能清单、项目结构和接管说明
